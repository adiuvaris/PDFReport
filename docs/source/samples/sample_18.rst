Sample 18

It is possible to define different text styles for the different row types in a table.
Any attribute of the text styles can be changed as needed.

Here the subtotal lines use font family ‘Courier’ and the total lines use ‘Times’.
The detail rows have a blue font color and the header row uses a font size of 11 points
